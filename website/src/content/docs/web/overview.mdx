![Обзор](./img/overview.png)

Веб-интерфейс позволяет вызывать функции Hestia, используя удобный интерфейс
прямо в браузере. С его помощью вы также можете обращаться к Hestia через API. 

Демо веб-интерфейса доступно [здесь](https://hestia-web-zhdo.onrender.com/).
