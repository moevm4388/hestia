Web-интерфейс Hestia поставляется в виде
[OCI-образа](https://github.com/moevm4388/hestia-web/pkgs/container/hestia-web),
который можно запустить с помощью [Docker](https://www.docker.com/) или
[Podman](https://podman.io/).

Вы также можете [запустить сервер вручную](#запуск-вручную).

## Docker/Podman

<Steps>

1.  Скачайте образ
    <Tabs syncKey="oci-platform">
        <TabItem label="Docker">
            ```sh
            docker pull ghcr.io/moevm4388/hestia-web:latest
            ```
        </TabItem>
        <TabItem label="Podman">
            ```sh
            podman pull ghcr.io/moevm4388/hestia-web:latest
            ```
        </TabItem>
    </Tabs>


2.  Запустите контейнер
    <Tabs syncKey="oci-platform">
        <TabItem label="Docker">
            ```sh
            docker run -p 8000:80 ghcr.io/moevm4388/hestia-web:latest
            ```
        </TabItem>
        <TabItem label="Podman">
            ```sh
            podman run -p 8000:80 ghcr.io/moevm4388/hestia-web:latest
            ```
        </TabItem>
    </Tabs>

3.  Откройте http://localhost:8000 в вашем браузере

</Steps>

:::tip[Совет]

Вы можете изменить порт, на котором запускается контейнер. Для этого укажите
его в команде:

<Tabs>
    <Tabs syncKey="oci-platform">
        <TabItem label="Docker">
            ```sh /<порт>/
            docker run -p <порт>:80 ghcr.io/moevm4388/hestia-web:latest
            ```
        </TabItem>
        <TabItem label="Podman">
            ```sh /<порт>/
            podman run -p <порт>:80 ghcr.io/moevm4388/hestia-web:latest
            ```
        </TabItem>
    </Tabs>
</Tabs>

:::

## Запуск вручную

<Steps>

1.  Склонируйте репозиторий
    ```sh
    git clone --depth 1 git@github.com:moevm4388/hestia-web.git
    ```

2.  Перейдите в директорию репозитория
    ```
    cd hestia-web
    ```

3.  Создайте виртуальное окружение Python
    <Tabs syncKey="operating-system">
        <TabItem label="Windows">
            ```powershell
            python -m venv venv
            .\venv\Scripts\activate
            ```
        </TabItem>
        <TabItem label="Linux/macOS">
            ```sh
            python3 -m venv venv
            source venv/bin/activate
            ```
        </TabItem>
    </Tabs>
    
4.  Установите зависимости
    ```sh
    pip install -r requirements.txt
    ```

5.  Запустите сервер
    ```sh
    fastapi run
    ```

6.  Откройте http://localhost:8000 в вашем браузере

</Steps>
